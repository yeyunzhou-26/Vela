# Vela Application UI

## Layout Contract

Vela uses Focused Workbench:

- Top Command Bar
- Left Mission Rail
- Center Mission Workspace
- Right collapsed Intelligence Spine
- Bottom-center Vela Voice Layer

## Hard Rules

- The Intelligence Spine is collapsed by default.
- The Mission Workspace shows one primary thing at a time.
- Do not make the center a dashboard.
- Do not use nested cards.
- Do not overuse purple, gradients, or decorative glow.
- Prefer native desktop density and calm hierarchy.

## Visual Direction

Calm Command Futurism:

- Graphite base
- Cool white text
- Cyan and amber accents
- Minimal purple
- Hairline dividers
- Stable dimensions
- Strong alignment

## First-Screen Test

Within two seconds, the user should know:

- Which mission is active
- What the main work surface is
- What the next step is
- Where to speak or type
- Where to inspect context, tools, guard, and review if needed
