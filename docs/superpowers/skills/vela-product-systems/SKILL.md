# Vela Product Systems

## Product North Star

Vela is an AI Operating Desk: a mission-first personal work system with voice, memory, agents, tools, permissions, and review.

It is not a chat app. Chat is one input mode inside the Mission Workspace.

## Fixed Product Decisions

- Product name: Vela
- Category: AI Operating Desk
- First screen: Mission Workspace
- Main layout: Focused Workbench
- Right side: collapsed Intelligence Spine by default
- Voice: system-level Vela Voice, not a mic button

## Mission Model

Every serious task should become a mission with:

- Goal
- Plan
- Steps
- Inputs
- Artifacts
- Tool calls
- Permissions
- Memory references
- Review result
- Recovery actions

## Decision Heuristics

- If a feature helps the user understand or complete the current mission, consider it.
- If a feature mainly demonstrates capability, hide it behind the Spine or defer it.
- If the center gets crowded, move secondary information to the Spine.
- If autonomy increases, increase inspectability and permission clarity.
